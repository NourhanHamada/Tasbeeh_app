Tasbeeh Mobile App

Description:

- Tasbeeh is a mobile application that includes an electronic counter for counting Tasbeeh and Thikr, daily life adhkar, such as morning and evening adhkar, and other random Islamic supplications. The app requests the user's location to fetch prayer timings according to their location. It also includes a light and dark mode and displays Gregorian and Hijri calendars.

Features: 

- Electronic counter for Tasbeeh and Thikr
- Daily life adhkar and Islamic supplications
- Fetches prayer timings according to the user's location
- Light and dark mode
- Displays both Gregorian and Hijri calendars

Requirements:
- Flutter development environment.
